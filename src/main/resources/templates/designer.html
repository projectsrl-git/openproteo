Cheat sheet

Executors
.ps1 → powershell · .bat/.cmd → cmd · .jar → java -jar (or force with Executor)
Builtin vars
${feedId} ${runId} ${runDate} ${landingIn} ${landingOut} ${stepDir} ${stepId} ${stepName} ${sharedDir} ${dir.<stepId>}
Output vars
script prints:  ##VAR name=value
Condition ops
== != >= <= > < contains notcontains matches exists notexists
Combine
clause && clause  or  clause || clause (not mixed)
Gate targets
a node id, END:SUCCESS, END:SKIPPED, END:REJECTED, END:FAILED, or (next node)
the saved XML is re-validated with the runtime parser before being written
