Variables

Manage variables that are common to all workflows, and edit the variables (and step parameters) of one or many workflows at once. Every change is validated against the workflow parser before anything is written.

Global variables (all workflows)

 

File-based (editable) — usable as ${name} in every workflow. Per-workflow variables of the same name win.

Workflow variables

Pick workflows with the three filters below (multi-select, they narrow each other). Select a single workflow to edit all its variables and step parameters; select several to edit only the variables they have in common.

All modified workflow XML files are validated before any of them is saved.
