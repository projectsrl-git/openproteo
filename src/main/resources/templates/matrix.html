Variables matrix

Every feed on a row, every variable on a column — edit the whole set in one sheet. Type in a cell to change a value; typing into an empty cell creates that variable for that feed. Only the cells you touch are saved.

Highlighted cells are unsaved edits. Empty cell = the variable is not defined for that feed (placeholder “not set”). Use ▾ in a column header to copy a value down to every visible feed. Arrow keys / Enter move between cells; pasting a block from Excel fills cells to the right and below.
