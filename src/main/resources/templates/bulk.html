OPENPROTEO Bulk create

Bulk create workflows from CSV

Pick a template workflow, paste two CSVs that carry everything, and generate many workflows at once. Column names are configurable, so you can paste your CSV as-is (extra columns are ignored). The feeds CSV sets feed metadata and may carry the dataschema/displayschema JSON inline; the tables CSV maps each feedId to its SQL table. Existing files are skipped unless overwrite is on.
