Import workflows

Upload a ZIP exported from OpenProteo. Workflows are created, or updated when a feed id already exists here. Before writing, retune the variables shared by the selected workflows and — unlike the Variables page — change the targetId and the production flag to move a feed from test to prod. Every workflow is validated by the parser before anything is written; nothing is imported if any fails.

The ZIP is only staged; nothing is written until you press Import.

Edit & import

Pick workflows with the three filters (multi-select, they narrow each other). One selected = edit everything; several selected = edit only what they have in common. With no filter, all workflows in the package are selected.
