Workflow tags: editable in designer, searchable column on home

Add free-form tags to a workflow, shown and searchable on the home dashboard.

- model/def/WorkflowDef + dto/WorkflowDto: new tags list.
- parser/WorkflowXmlParser + parser/WorkflowXmlWriter: tags are stored on the
  root <workflow> element as a comma-separated "tags" attribute.
- web/ApiController.toDto: carry tags into the designer DTO.
- templates/designer.html: new "Tags" field in the workflow settings
  (comma-separated, ${var} placeholders allowed); loaded and saved with the rest.
- web/PageController + templates/dashboard.html: new "Tags" column on the home
  list; placeholder tags are resolved with the workflow variables for display,
  and tags are included in the row search (filter by feed id, name, schedule or
  tag).

Compiles (103 classes); designer/dashboard scripts pass node --check.
