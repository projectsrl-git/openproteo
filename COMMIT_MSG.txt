Operations: weather icons per source row (fancy)

Adds a "Sky" column to the per-source rollup table in the Operations overview: a
weather emoji summarising each source's feeds.

- all successful          -> shining sun
- some failed             -> storm
- all failed              -> heavy thunderstorm (meaner)
- all still to run        -> white clouds
- success + to-run only   -> sun behind cloud
- running (my choice)     -> sun behind small cloud
- on hold (my choice)     -> fog
- aborted / mixed         -> sun behind rain / sun behind cloud

Any-failed takes priority (storm), then all-success, then all-to-run, then the
success+to-run mix, then running / on-hold / aborted / mixed. Each icon carries a
title tooltip with counts and a compact legend sits under the table. Emojis are
written as \uXXXX escapes (proxy-safe, consistent with the codebase).
