CSV viewer: show displayschema DisplayName in the column header

Friendlier CSV view: each header shows the human DisplayName from the feed's
displayschema.json (matched to the CSV column = ColumnName), with the technical
ColumnName kept underneath as a sub-label + tooltip.

- ApiController.displayNameMap(feedId) reads <feedDir>/displayschema.json
  (list or under "columns") into ColumnName -> DisplayName (keys name/ColumnName/
  COLUMN_NAME; labels DisplayName/displayName/display_name). csv/meta now returns
  displayNames aligned to columns (exact match, then case-insensitive). Empty for
  shared files / feeds without a schema.
- viewer.js CSV header renders DisplayName (bold) + ColumnName (sub); range filter
  dropdown shows "DisplayName — ColumnName"; chips use the DisplayName. Falls back
  to ColumnName when absent. Sort/resize/paging unchanged.

Compiles (103 classes). viewer.js verified.
