safecopy executor (exec="safecopy"):
- Copies wildcard-matched files from an input directory to an output
  directory, writing each file as <name>.on_fly_ and renaming it to the
  final name only after the copy completes (atomic move when possible).
  A downstream process watching the landing zone never sees a partial
  file. Params: source, dest, pattern, tmpSuffix (default .on_fly_).
  Outputs matchedFiles / matchedCount / bytesCopied. Tested.
