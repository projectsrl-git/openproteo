Operations board: live executions + per-source rollup

New /overview page (linked from the dashboard) with two auto-refreshing views.

- Executions in progress (3s): GET /api/overview/active from engine.queueSnapshot
  -- every queued/running/waiting run across all feeds with source, status,
  current step, progress, an Open link and a Stop button.
- By source rollup (20s + manual): GET /api/overview/rollup reads each feed's
  latest run once and buckets feeds per source into not-run / running / success /
  failed / aborted / other (running detected via engine.activeRunId, so active
  feeds skip the disk read). Page shows totals tiles + a per-source table with a
  stacked mix bar.
